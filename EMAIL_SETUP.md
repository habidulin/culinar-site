# Настройка отправки Email

## Шаги для настройки:

### 1. Зарегистрируйтесь на Resend
- Перейдите на https://resend.com
- Создайте бесплатный аккаунт (3000 писем/месяц бесплатно)

### 2. Получите API ключ
- После регистрации перейдите в https://resend.com/api-keys
- Нажмите "Create API Key"
- Скопируйте ключ (он показывается только один раз!)

### 3. Добавьте ключ в проект
- Откройте файл `.env.local` в корне проекта
- Замените `your_resend_api_key_here` на ваш настоящий API ключ
- Пример: `RESEND_API_KEY=re_123abc456def789`

### 4. Верифицируйте домен (опционально, но рекомендуется)
- В Resend перейдите в "Domains"
- Добавьте ваш домен и следуйте инструкциям
- Пока домен не верифицирован, письма будут отправляться с `noreply@resend.dev`

### 5. Перезапустите сервер
```bash
npm run dev
```

## Готово! 🎉

Теперь форма на сайте будет отправлять письма на **culinar.potsdam@gmx.de**

## Альтернатива: Gmail SMTP (если не хотите использовать Resend)

Если хотите использовать Gmail вместо Resend:

1. Установите nodemailer:
```bash
npm install nodemailer
```

2. В `.env.local` добавьте:
```
EMAIL_USER=ваш_gmail@gmail.com
EMAIL_PASS=ваш_app_password
```

3. Замените содержимое `src/app/api/contact/route.ts` на код с nodemailer (могу предоставить по запросу)

## Тестирование

После настройки:
1. Откройте сайт
2. Перейдите в раздел "Kontakt"
3. Заполните форму
4. Нажмите "Nachricht senden"
5. Проверьте почту culinar.potsdam@gmx.de
